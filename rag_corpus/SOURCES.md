# Fuentes del corpus RAG

El corpus se especializa en **FastAPI** (framework web de Python). Los documentos
se redactaron a partir de la documentación oficial de FastAPI.

| Archivo | Tema | Fuente oficial |
|---------|------|----------------|
| 01_first_steps.md | App mínima, path operations, OpenAPI | fastapi.tiangolo.com/tutorial/first-steps |
| 02_request_body_pydantic.md | Request body con modelos Pydantic | fastapi.tiangolo.com/tutorial/body |
| 03_dependencies.md | Inyección de dependencias (`Depends`) | fastapi.tiangolo.com/tutorial/dependencies |
| 04_response_model.md | `response_model`, filtrado de salida | fastapi.tiangolo.com/tutorial/response-model |
| 05_testing_and_errors.md | `TestClient`, `HTTPException`, status codes | fastapi.tiangolo.com/tutorial/testing y /handling-errors |
| 06_project_structure_routers.md | `APIRouter`, estructura de proyecto | fastapi.tiangolo.com/tutorial/bigger-applications |

Los textos fueron sintetizados/abreviados de la documentación oficial (repositorio
`fastapi/fastapi`, carpeta `docs/en/docs`) con fines educativos para este TP.
